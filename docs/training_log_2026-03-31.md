## 1. 데이터 수집

### 1-1. MGnify (실제 데이터)

유럽 생물정보학 연구소(EBI)에서 운영하는 MGnify에서 토양 미생물 데이터를 수집했다.

**수집 과정:**
- MGnify API로 토양(soil) 바이옴 관련 연구를 검색 → **626개 연구** 발견
- 그 중 100개 분석(analysis)의 BIOM 파일을 다운로드
- 각 BIOM 파일에서 미생물 이름과 개체 수(count)를 추출
- 샘플당 약 18초 소요, 총 약 28분

**수집 결과:**
- 샘플 수: **100개**
- 발견된 미생물 종류: **975종**

**수집 중 겪은 문제와 해결:**

처음에 MGnify의 taxonomy API 엔드포인트(`/taxonomy/ssu`)를 사용했는데, "Bacteria" 같은 최상위 분류만 반환하고 구체적인 미생물 이름(속 수준)을 주지 않았다. 파이프라인 v5.0의 API 응답이 상위 수준까지만 제공하는 것으로 확인.

해결: BIOM 파일(`*_SSU_OTU_TABLE_JSON.biom`)을 직접 다운로드해서 파싱하는 방식으로 전환. BIOM 파일 내부의 `rows[].metadata.taxonomy` 필드에서 `g__`(genus) 또는 `f__`(family) 접두사가 붙은 분류를 추출하여 성공.

추가로, taxonomy 필드가 리스트가 아닌 세미콜론(`;`)으로 구분된 문자열인 경우가 있어서, 문자열/리스트 양쪽 모두 처리하도록 파서를 수정했다.

3. 전체 81개 사이트 중 47개에서 토양 미생물 + 화학 데이터가 있는 것을 확인

## 3. 모델 학습

### 3-1. 학습 방식: 인과적 언어 모델링 (Causal LM)

각 토양 샘플의 미생물 목록을 풍부도 순서대로 나열하고, "앞에 나온 미생물들을 보고 다음에 올 미생물을 맞추기" 방식으로 학습한다.

GPT가 "오늘 날씨가 ___"에서 다음 단어를 예측하듯, 이 모델은 "Mycobacterium, Bryobacter, ___"에서 다음에 올 미생물을 예측한다.

이 방식으로 모델은 다음을 자연스럽게 학습하게 된다:
- 같이 잘 나타나는 미생물 쌍 (공생 관계)
- 같이 안 나타나는 미생물 쌍 (길항 관계)
- 특정 환경 조건을 암시하는 미생물 조합 (바이옴 특성)

## 4. 모델 추론 테스트

best.pt 모델(에폭 8)을 사용하여, 미생물 이름 몇 개를 넣고 "같이 나올 법한 미생물"을 예측하는 테스트를 수행했다.

### 테스트 1: 산성 토양 미생물로 시작

**입력:** Mycobacterium, Bryobacter, Acidothermus

**모델 예측:**
1. Burkholderiaceae — 산성 토양에 서식하는 세균과
2. Candidatus_Koribacter — 산성 토양 전문 세균 (Acidobacteria 문)
3. Candidatus_Solibacter — 산성 토양에서 흔한 세균
4. Micromonosporaceae — 토양 방선균

**해석:** 산성 토양 미생물(Bryobacter, Acidothermus)을 넣자 같은 산성 토양에 사는 Candidatus_Koribacter, Candidatus_Solibacter를 예측했다. 이들은 모두 Acidobacteria 문에 속하는 산성 토양 전문 세균이다.

### 테스트 2: 질소 관련 미생물로 시작

**입력:** Haliangium, Caulobacteraceae, Nitrosomonadaceae

**모델 예측:**
1. Mycobacterium — 가장 흔한 토양 세균
2. Xanthobacteraceae — 질소 고정 관련 세균과
3. Gemmataceae — Planctomycetes 문
4. Rhodoplanes — 질소 순환에 관여하는 광합성 세균

**해석:** 질소 순환 관련 미생물(Nitrosomonadaceae)을 넣자 Xanthobacteraceae(질소 고정)와 Rhodoplanes(질소 순환) 등 질소 관련 미생물이 예측에 포함되었다.

### 테스트 3: 흔한 토양 미생물로 시작

**입력:** Candidatus_Solibacter, Acetobacteraceae

**모델 예측:**
1. Pajaroellobacter
2. Bryobacter — 산성 토양에서 흔한 세균
3. Ktedonobacteraceae — 토양 방선균
4. Mycobacterium — 가장 흔한 토양 세균
5. Gemmatimonadaceae — 건조한 토양에서 흔한 세균

**해석:** 범용적인 토양 미생물을 넣자 다양한 토양 환경에서 흔히 발견되는 미생물들이 고르게 예측되었다.

### 추론 테스트 한계

- 같은 미생물 이름이 반복 출현하는 경향 (데이터 100개의 다양성 부족)
- 데이터가 적어서 드문 미생물 간의 관계는 학습하지 못함
- 데이터를 500~1000개 이상으로 늘리면 예측 다양성과 정확도 개선 예상
