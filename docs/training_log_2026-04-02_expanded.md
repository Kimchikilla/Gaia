## 3. 성능 하락 원인

어휘를 확장했지만 성능이 오히려 하락한 이유:

1. **새 토큰 2,970개의 임베딩이 랜덤 초기화** — 아직 의미를 학습하지 못함
2. **학습 데이터 부족** — 3,624개로 12,639개 어휘를 학습하기엔 부족
3. **에폭 부족** — 5 에폭으로는 새 임베딩이 수렴하지 못함

## 4. 해결 방안

- MGnify에서 **10,000개 이상** 추가 수집하여 총 13,000개+ 데이터 확보
- 더 많은 에폭으로 학습 (10~20 에폭)
- 새 토큰의 임베딩만 먼저 학습 → 전체 미세조정 2단계 접근

## 5. 디스크 정리

- Naylor fastq 원시 파일 삭제: 7.7GB 절약
- 프로젝트 전체 용량: 10.6GB → **3.4GB**
