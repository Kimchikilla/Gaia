## 4. 분석

### RF가 USDA 데이터에서 이긴 이유

1. USDA 데이터의 미생물 이름 형식이 MGM 어휘와 매칭이 낮음
2. USDA 데이터가 423개로 적어서 사전학습의 이점이 줄어듦
3. RF는 1,591종 전부 사용, Gaia는 매칭된 일부만 사용

### Gaia의 강점이 드러나는 조건

- 데이터가 많을 때 (MGnify 2,887개 → 바이옴 분류 승리)
- MGM 어휘 매칭이 높을 때 (가뭄 탐지 91.9%)
- RF가 할 수 없는 과제 (풍부도 복원)

### pH R²=0.71의 의미

- 미생물 조합만으로 토양 pH를 R²=0.71 수준으로 예측 가능
- 이전 가짜 라벨 R²=0.22와 비교하면 실제 데이터가 훨씬 유의미
- $50 화학 분석 없이 미생물 시퀀싱만으로 pH 추정 가능성 확인
