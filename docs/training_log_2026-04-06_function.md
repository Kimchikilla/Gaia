# Gaia 기능 프로필 분석 — 2026년 4월 6일

---

## 1. 방법

문헌 기반 기능 annotation으로 토양 미생물의 기능 그룹을 정의하고,
Naylor 가뭄 실험 데이터(drought 303, control 320)에서 기능별 변화를 분석.

## 2. 결과: 가뭄 vs 정상 토양의 기능 차이

| 기능 | 정상 | 가뭄 | 변화 |
|------|------|------|------|
| Carbon Decomposition | 2,146 | 4,262 | **+98.6%** |
| Pathogen Suppression | 2,737 | 4,581 | **+67.4%** |
| Methane Cycling | 14 | 20 | +50.8% |
| Nitrification | 366 | 420 | +14.7% |
| Phosphorus Solubilization | 1,664 | 1,877 | +12.8% |
| Plant Growth Promotion | 3,394 | 3,128 | -7.8% |
| Nitrogen Fixation | 900 | 779 | **-13.4%** |
| Organic Matter Decomposition | 3,842 | 3,319 | -13.6% |
| Denitrification | 788 | 578 | **-26.6%** |

## 3. 해석

가뭄이 오면:
- 탄소 분해 미생물이 급증 (+98.6%) → 토양 탄소 손실 위험
- 질소 고정 능력 저하 (-13.4%) → 작물 영양 공급 감소
- 탈질소화 감소 (-26.6%) → 질소 순환 교란
- 병원균 억제 미생물 증가 (+67.4%) → 스트레스 방어 반응

기존 논문 결과와 일치하며, 미생물 기능 분석이 토양 건강 진단에 활용 가능함을 확인.

## 4. Gaia의 기능 분석 능력

현재:
- 문헌 기반 기능 매핑 (annotation)
- 미생물 이름 → 알려진 기능 연결

향후:
- PICRUSt2 기능 유전자 예측 (설치 완료, 대규모 데이터 처리 필요)
- 샷건 메타게놈으로 직접 기능 확인
- 모델 임베딩에서 미지의 미생물 기능 추론
